Pentaho Saiku Sidekicks

Quick Start

About

This extension intent is to provide a set of features to improve productivity working with Pentaho Business Intelligence.

Currently, there is only one feature: Heat maps. Using Saiku Analytics plug-in, sometimes you need to analyze the reports in a heat map perspective. Heat maps allow insights of data giving visual cues about how data varies, where anomalies or highest/lowest values are concentrated.

This plugin adds heat map feature to Saiku reports inside Pentaho. Just by right-clicking the report, you can produce a heat map for each row, for each columns or even the entire table.

Please, visit the extension web site to ask for new features or to submit bugs! http://github.com/natenho/pentaho-sidekicks
